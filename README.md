#The Lipstcik Effect in Europe

##What is the Lipstick Effect?
The lipstick effect is a phenomenon we can admire and which is produced through the 
realtionship between the consumption of affordable luxury and times of economic 
distress.  

##What does the project do?
With this project we analyse and compare cosmetic sales in Spain, France and Italy 
thoughout the years (2003-2023). This data is compared with unemployment rates in the same
countries in order to test whether if the lipstick effect actually happens and if we can see
it refelcted in these datasets. 

##Data sources
- Cosmetics retailsales: Eurostat (dataset: sts_trtu_m, category G47_NF_HLTH) 
- Unemployment rates:Eurostat (dataset:une_rt_m)

##Outcomes
- Even during periods in which unemployment peaked, sales in the cosmetic industry 
remained unstable 
- During 2020 (COVID) we do see a small dip which is rapidly recovered afterwards
- In general the lipstick effect is proven by the data, as we see how sales in cosmetics remain
unaffected when it comes to economic crises
